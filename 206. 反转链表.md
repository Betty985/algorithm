# 迭代
两个指针移动过程中改变指向，在此之前需要**保存当前节点的下一节点**,否则会丢失
- 节点不为空时，循环
- next保存当前节点的下一个节点
- cur(当前节点）指向前一个节点
- pre(前一个节点）赋值为cur
- cur赋值为next
```js
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    let pre=null
    let cur=head
    while(cur){
        let next=cur.next
        cur.next=pre
        pre=cur
        cur=next
    }
    return   pre
};
```
